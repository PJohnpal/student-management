# Student Management System

A comprehensive student management system built with React.js, FastAPI, and MySQL.

## Features

- **User Management**: Student, Teacher, and Admin roles
- **Course Management**: Create and manage courses
- **Enrollment System**: Student course enrollment
- **Grade Management**: Track and manage student grades
- **Dashboard**: Overview with statistics and recent activities
- **Authentication**: JWT-based secure authentication

## Tech Stack

- **Frontend**: React.js, React Router, Axios
- **Backend**: FastAPI, Python, SQLAlchemy
- **Database**: MySQL
- **Authentication**: JWT Tokens
- **Deployment**: Docker & Docker Compose

## Quick Start

### Prerequisites

- Docker and Docker Compose
- Node.js 18+ (for local development)
- Python 3.11+ (for local development)
